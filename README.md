# DaDa Taxi - Driver Edition

## 項目概述

**DaDa Taxi** 是一個運輸平台司機端應用，支持出租車、貨車和私家車服務。

- **功能按鈕文字**: 白色

## 功能模塊

### 1. 登入/註冊系統
- 電郵密碼登入
- Google 電郵快速登入
- 多語言選擇 (國際化支持)
- 服務國家/地區選擇

### 2. 註冊流程
**步驟 1**: 基本資料
- 界面語言選擇
- 姓名、電郵、電話
- 密碼設定

**步驟 2**: 服務地區
- 選擇 Uber 國際服務國家/地區

**步驟 3**: 車輛信息
- 車輛類型 (的士/貨車/私家車)
- 車輛型號
- 車牌號碼
- 邀請碼 (可選)

**步驟 4**: Stripe KYC 認證
- 身份驗證
- 駕駛執照檢查

## 私隱政策要點

1. 收集信息: 姓名、電話、電郵、車輛信息、位置數據
2. 信息用途: 提供服務、支付處理、身份驗證
3. 信息共享: Uber API、Stripe、相關部門
4. 數據安全: 行業標準加密
5. 用戶權利: 訪問、更正、刪除個人信息

## 運營模式

- 訂閱制收費 (無廣告)
- 廣告收益維持運營 (免費/低價方案)
- 未與 Uber 分成 (用戶達標後再洽談)
